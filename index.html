About

This is a simple application for converting images. In fact, it is a simple draft of an application, merely a PoC aiming to show that it is possible to quickly wire up Qt and ImageMagick to perform simple image manipulation. The application was written in a day or so without any prior knowledge about utilised frameworks/libraries so they might not have been used properly. By "properly" I mean, in accord with patterns and/or idioms accepted by the communities of the respected libraries.

Currently, the application implements two features:

  • image resizing
  • conversion to JPG

However, more features are planned (e.g. conversion to other formats, simple filters and/or distortions, etc.).

Built upon

As mentioned, the application is built on top of the two most popular (in their respective domains) libraries, i.e. ImageMagick and Qt. It has been tested on Microsoft Windows 7 (x64) and Mac OS X Maverick & Yosemite.

Release

Unfortunately, due to lack of time I have not published binary version of the application. Thus, if anyone would like to use it one has to install Qt and ImageMagick development packages and build the application from source code himself.

I plan to release binaries of the first version ("the draft"), most probably with statickally linked dependencies, soon; however, I can't give any specific date. The plan is to make release for MS Windows, Mac OS X and Linux (probably deb for Ubuntu).
